Чтобы бот работал нужно установить последнюю версию Python с официального сайта
https://www.python.org/

Далее открыть консоль(cmd) и вставить по порядку следующие команды:
pip install pyTelegramBotAPI
pip install win10toast
pip install os-win

Чтобы бот реагировал на ваши действия нужно:
1) В файл token.txt вставить ваш api_token без пробелов и других прочих символов
2) В файл username вставить ваше имя записанное в вашем телеграм аккаунте

Чтобы добавить бота в автозапуск нужно:
1) Нажать сочетание клавиш Win+R
2) В открытое меню вставить shell:startup
3) В открытую папку перенести файл RuFa 1.11.00
После данных действий папки можно закрыть

ВНИМАНИЕ
Настоятельно рекомендую не спамить командами Shutdown и Lock т.к. это может привести к необратимым последствиям.

Приятного пользования!
